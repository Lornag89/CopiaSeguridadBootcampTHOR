¡IMPORTANTE! NORMAS:

  1. NUNCA COMBINÉIS EN NINGÚN CASO LAS CLASES .container, .row y .col
    • <span class="container row"> -> ¡NO!
    • <span class="row col"> -> ¡NO!
    • <span class="container col"> -> ¡NO!
  2. NUNCA PONGÁIS CLASES QUE APORTEN ESTILO A NINGÚN .container, .row y .col
    • <span class="container bg-warning"> -> ¡NO!
    • <span class="row border border-2"> -> ¡NO!
    • <span class="col shadow bg-info"> -> ¡NO!
  3. La forma correcta SIEMPRE será crear un div nuevo que tenga los estilos a parte o un div nuevo para hacer un nuevo row, etc.
